Global Service Catalogue - Invalid URL Checker


This tool is used to verify that certain user-entered URLs contained in the SERVIR Global Service Catalogue are valid and reachable via the internet. Specifically, entries in the DATA, TOOLS, NEWS, and TRAINING MATERIALS sections within each service are checked for valid accessibility. The tool executes a Python script (via Django framework) that calls the Service Catalogue API to retrieve info from all services. Invalid URL entries are gathered and reported in a table on the results page that is displayed after the tool has finished processing. If needed, the resulting table can be exported to a comma separated values (.csv) file - which can be opened in Excel.

A secondary check is also made on URLs specific to the DATA section within each service to ensure that all of the entries link to a particular data item within the SERVIR Global Data Catalogue (e.g. http://gis1.servirglobal.net:8080/geonetwork/...). DATA URL entries found pointing to sites other than the Global Data Catalogue are reported in a second table on the results page.
